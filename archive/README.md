# Archive

This directory contains **legacy, non-production artifacts** retained for
historical reference.

## What is here

- `archive/examples/`: early proof-of-concept SAM templates and handlers that
  are no longer part of the supported mnemospark backend surface.

## Active production surface

Use these paths for all current development and operations work:

- `template.yaml` (main SAM stack)
- `services/` (live Lambda handlers)
- `docs/` (OpenAPI and endpoint documentation)
- `tests/` (active automated tests)
